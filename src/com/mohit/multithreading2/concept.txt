  class test{

          synchronized (lock) {
              System.out.println("here in outer block ");
              synchronized (lock) {
                  System.out.println("again inner block");
              }
          }
      }
  }

  this proves ki synchronized is also reenterant. It means same thread can re-acquire the lock but not different thread.
